# 🛒 ShoppingList App

A simple, modern Shopping List app built with **Jetpack Compose**!  
You can easily add, edit, and delete shopping items with a clean UI.

---

## 🚀 Features

- ✅ Add shopping items with name & quantity  
- ✏️ Edit existing items inline  
- 🗑️ Delete items from the list  
- 🎨 Jetpack Compose UI for a modern look  
- 💾 Uses `mutableStateOf` for real-time state management  

---

## 🧱 Built With

- [Kotlin](https://kotlinlang.org/)
- [Jetpack Compose](https://developer.android.com/jetpack/compose)
- [Material3](https://m3.material.io/)
- Android Studio
